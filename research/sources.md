# Sources — Cold Outreach Pipeline for B2B SaaS

Ten practitioners who **build and run outbound for a living**, not commentators who
only write about it. Selection criteria:

1. **Practitioner proof** — each has personally built pipeline, run an outbound
   agency/tooling company, or trained thousands of reps (receipts noted below).
2. **Collectable on two channels** — active on LinkedIn *and* a podcast / YouTube /
   newsletter, so both `linkedin-posts/` and `youtube-transcripts/` can be populated.
3. **Recency** — all publishing actively in 2025–2026 (dated items cited).
4. **Non-obvious but high-signal** — deliberately skips generic "top sales influencer"
   listicles in favor of operators respected inside the outbound community.

Coverage is intentionally spread across the whole pipeline: cold calling, cold email,
multichannel sequencing, signal-based targeting, personalization, and SDR operations.

Captured content lives in [`linkedin-posts/`](./linkedin-posts) and
[`youtube-transcripts/`](./youtube-transcripts); a foundational reference sits in
[`other/`](./other).
